# Anvaye — E-Commerce Platform

> One stop shop for your wish-list

Anvaye is a modern e-commerce platform connecting buyers with verified sellers. Features include secure payments, order tracking, verified reviews, and a premium subscription model.

## ✨ Features

- **Buyer Journey**: Browse sellers, view products, place orders with UPI
- **Seller Dashboard**: Manage products, track orders, view analytics
- **Subscription Model**: Anvaye Premium unlocks unlimited seller access
- **Order Tracking**: Real-time order status updates
- **Verified Reviews**: Only actual buyers can leave reviews
- **Dark Mode**: Automatic theme switching based on system preference
- **Progressive Web App**: Installable on mobile devices
- **Responsive Design**: Works on all screen sizes
